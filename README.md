# 海外服务使用指南

面向普通用户的海外数字服务教程知识库，主要整理 Google 账号、外区 Apple ID、ChatGPT / Gemini 订阅、礼品卡和常见工具使用流程。

项目定位是公开教程站：只做公开信息整理、个人经验复盘、流程说明和风险避坑，不提供后台系统、交易撮合或代操作服务。

## 内容边界

本站允许整理：

- 官方入口、公开活动和常见页面路径；
- 账号注册、地区设置、订阅支付等正常流程说明；
- 礼品卡、App Store、Google One 等公开页面的风险提示；
- 工具使用经验、截图说明和常见问题排查；
- 如何保护账号、支付信息、验证码和订单信息。

本站不提供、也不鼓励：

- 代充、接码、黑卡、盗刷、账号交易、验证码交易；
- 规避风控、绕过平台规则、异常注册、异常支付；
- 售卖来路不明账号、卡密、订阅或兑换码；
- 要求用户提交密码、验证码、2FA 密钥或支付信息的第三方操作教程；
- 对任何第三方平台、商家、机器人或交易服务做强推荐或成功承诺。

## 教程状态

- `可用`：流程已经整理并可公开阅读；
- `待完善`：内容仍在补充或截图不完整；
- `高风险提示`：涉及订阅支付、礼品卡、账号安全或第三方平台，需要重点阅读风险说明；
- `不推荐`：保留为风险说明或避坑记录，不建议普通用户操作。

## 本地查看

直接打开 `index.html` 即可查看。新增或替换教程图片后，可以运行：

```bash
node tools/generate-webp.js
```

脚本会在保留原图的同时生成 WebP 派生图，页面会优先加载 WebP，点击预览时仍可回到原图。
